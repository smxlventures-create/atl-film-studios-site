SET 07 — Precinct & Law

Prison Cell & Visitation

A high-security penitentiary cell and visitation booth.

About the Set

Our high-security Prison Cell set features a detailed, concrete-textured cell block with heavy iron bars, a small high window, a concrete-slab bunk bed, and an integrated visitation room. The visitation booth is fully dressed with plexiglass dividers, wall-mounted steel telephones, and stools, allowing for highly emotional and dramatic communication scenes. Ideal for cinematic feature films and gritty documentaries.

Key Features Included

  • ✓ High-Security Iron Bars
  • ✓ Plexiglass Visitation Booth
  • ✓ Steel Wall-Mounted Phones
  • ✓ Concrete-Slab Bunk
  • ✓ Concrete Textured Walls
