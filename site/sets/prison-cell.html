STANDING SET 07 · PRISON CELL

Prison Cell Set

A state or federal prison cell set. Featuring a heavy steel door with a barred window, concrete-style bunk, and distressed concrete walls.

Book on Peerspace Request Quote

Maximum security.

Our prison cell set is designed to look like a maximum-security state or federal prison cell. Ideal for dramatic incarceration sequences, prison breaks, and character dialogue.

The set features a heavy steel door with a working barred food slot and window. The interior contains a built-in concrete-style bunk, thin institutional mattress, and a metal sink/toilet combination unit. Distressed concrete-style walls complete the sterile, gritty look.

This set is perfect for long-term confinement scenes, dramatic monologues, and intense character interactions. The lighting can be customized to create sterile institutional lighting or high-contrast shadows through the door slot, adding visual depth to every frame.
